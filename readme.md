# Git & GitHub Practice Repository

This repository was created to practice and understand Git and GitHub fundamentals.

## What I Practiced

- Creating a Git repository
- Initializing Git
- Making commits
- Writing commit messages
- Creating and managing files
- Using `.gitignore`
- Pushing code to GitHub
- Updating existing files
- Working with branches (basic)
- Understanding repository structure

## Repository Contents

This repository contains sample files created while learning Git and GitHub. The files are for practice purposes only and do not represent a project.

## Purpose

The goal of this repository is to build confidence with Git commands and GitHub workflow before working on real-world projects.
